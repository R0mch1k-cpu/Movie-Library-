# Movie Library (Личная кинотека)

Автор: [Роман Васюрчик ]

## Описание
GUI-приложение на Python/Tkinter для управления личной кинотеатекой. Хранение фильмов в JSON, фильтрация по жанру/году, валидация ввода.

## Установка
1. Клонируйте репозиторий: `git clone <URL>`
2. Запустите: `python main.py`

## Использование
- Введите данные фильма, нажмите "Добавить фильм".
- Фильтруйте по жанру (комбобокс) или году.
- Данные сохраняются в `movies.json`.

Пример теста:
- Добавьте "Интерстеллар", жанр "Sci-Fi", 2014, 8.7.
- Отфильтруйте по "Sci-Fi".

## Технологии
- Python 3, Tkinter, JSON
